# Architecture Baseline (AWO/CRI)

This directory contains the **frozen architecture baseline** for the  
Aurora Workflow Orchestration (AWO) and Continuous Research Integration (CRI) system.

These documents define the **dependency graph**, **execution order**,  
and **governance-stable architecture** of the AWO/CRI ecosystem.  
They serve as the canonical reference for reconstructing the system  
from first principles.

All files in this directory are:

- **baseline / frozen**
- **architecture-level artifacts**
- subject to **Formal Change Requests (FCRs)** for modification
- awaiting metadata headers that will later be auto-generated by the metadata pipeline

No document in this directory should be edited directly without following ARI governance.

The full, DAG-aligned backlog of the AWO/CRI system.
Defines the execution tiers, task groupings, and correct ordering of work.
